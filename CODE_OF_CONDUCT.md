# Code of Conduct

## 我们的承诺

我们致力于创建一个开放、友好、包容和安全的社区。我们的社区欢迎来自任何背景和身份的参与者。

## 我们的标准

有助于创建积极环境的行为包括：

- 使用热情和包容的语言
- 尊重不同的观点和经验
- 接受建设性批评
- 关注什么对社区最有利
- 表现出对其他社区成员的同情

不可接受的行为包括：

- 使用令人反感的语言或图像，以及不受欢迎的性关注或騷擾
- 有意骚扰、威胁或恐吓
- 基于身份、性别、性取向、能力或经历的歧视
- 发布他人的私人信息（"doxxing"）
- 可能被合理地认为是不专业或不适当的其他行为

## 实施

社区领导者负责澄清和实施可接受行为标准，并将对任何他们认为不适当、威胁性、冒犯性或有害的行为采取适当的、公正的纠正措施。

社区领导者有权利和责任删除、编辑或拒绝与本行为准则不符的评论、承诺、代码、维基编辑、问题和其他贡献，并在适当时传达审核决定的理由。

## 范围

本行为准则适用于所有社区空间，也适用于在公共场所正式代表社区时的个人行为。

## 报告

如果你目睹或受到了不可接受的行为，请通过以下方式报告：

- 在 GitHub Issue 中私下报告（标记为 "Code of Conduct")
- 联系项目维护者

所有报告将被保密处理。

## 执行

违反行为准则的社区成员可能会面临临时或永久禁令。

感谢你帮助我们创建一个安全、热情的社区！
