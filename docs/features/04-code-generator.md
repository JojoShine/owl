# 代码生成器

## 功能说明

基于数据库表结构，动态生成完整的增删改查模块，无需编写任何代码，配置完成后即可直接使用。

---

## 使用流程

1. 在数据库中创建业务表（表名不使用 `owl_` 前缀）
2. 登录系统，进入「代码生成器」
3. 在表列表中找到目标表，点击「初始化配置」
4. 配置每个字段的展示方式、表单组件、是否可搜索等
5. 完成配置后，系统自动提供完整的增删改查功能，无需重启服务

---

## 字段配置项

| 配置 | 说明 |
|------|------|
| 显示名称 | 列头和表单 Label 显示的中文名 |
| 表单组件 | input / textarea / select / datepicker 等 |
| 是否在列表展示 | 控制该字段是否出现在表格列 |
| 是否可搜索 | 控制该字段是否出现在搜索栏 |
| 是否必填 | 控制表单校验 |

---

## 注意事项

- `owl_` 开头的系统表不允许通过代码生成器初始化
- 删除模块配置后，可重新初始化配置
- 适用于标准的增删改查场景，复杂业务逻辑建议手动开发
